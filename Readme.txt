Este es un proyecto realizado para la clase de Pensamiento Computacional Orientado a Objetos de primer semestre de 
ingeniería en Ciencia de Datos y Matemáticas. El proyecto consiste en un sistema en consola de un sistema de compras en el que
los compradores pueden comprar productos registrados por los proveedores si tienen dinero suficiente y existe en stock 
el producto.


-Cada usuario tiene una contraseña y un username individual. 
-Hay un menú para cada usuario, se pueden registrar productos y modificarlos, pero no eliminarlos. 
-Los datos son guardados en archivos csv, tanto los datos se los usuarios, como los productos y las compras que realicen 
los compradores.
-Cada input esta validado para evitar errores y fallos en el guardado de los datos.



Areas de mejora: Interfaz gráfica, registro de usuario, eliminar productos.



-Ejecución: 

Solo ejecuta el archivo mi_programa.exe en la misma carpeta que los csv para evitar conflictos en la lectura y escritura 
de datos.
